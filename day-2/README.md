# Day 2 - HTML Structure

This folder contains all HTML files for the Mobile Recharge App.

## Files:
- `login.html` - User login page
- `signup.html` - User registration page
- `dashboard.html` - User dashboard
- `recharge.html` - Mobile recharge page
- `subscriptions.html` - Subscription management
- `admin.html` - Admin dashboard
- `analytics.html` - Analytics page
- `offers.html` - Special offers page
- `index.html` - Landing page
- `test.html` - API testing page

All HTML files include inline CSS using Tailwind CDN.
