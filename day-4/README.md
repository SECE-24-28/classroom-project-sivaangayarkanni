# Day 4 - Validations and JavaScript

This folder contains HTML files with form validation and JavaScript functionality.

## Files:
- HTML files (login, signup, dashboard, recharge, subscriptions)
- `validation.js` - Form validation functions

## Validation Features:
- Email validation
- Phone number validation (Indian format)
- Password validation (minimum 8 characters)
- Amount validation (1-10000 range)
- Error display and clearing functions
- Form validation for login and recharge

All validations run on client-side before form submission.
